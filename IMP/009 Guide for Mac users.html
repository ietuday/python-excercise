In Mac,terminal will look something like

vignesh: ~$  

It is nothing but the default path like C:\user\vignesh>  in windows.

If you need to create new directory type mkdir just like in windows

you can change directory by typing cd just like in windows.

vignesh: ~$ cd Desktop
vignesh: Desktop$

Execution of file is also same like windows, just go to that directory and write python version and filename

vignesh: Desktop$ python3 hello.py
Hello world
vignesh: Desktop$
