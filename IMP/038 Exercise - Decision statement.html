Grade Converter

Write a program which gets input from the user and converts it into grade depending upon the mark scored.

Mark           Grade

90 - 100                S

80 - 90                  A

70 - 80                  B

60 - 70                  C

50 - 60                  D

40 - 50                  E

below 40               F

Example:

Enter the mark : 88
You have scored A grade
