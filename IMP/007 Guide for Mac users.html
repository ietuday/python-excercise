Python 2.x

  • Python 2x is installed in Mac as default. So just go head and search for Terminal and open it.
  • In terminal type python2 and press enter to open interactive python shell.
  • Type quit() to exit from python shell

Python 3.x

  • Download Python 3x from python.org.
  • In terminal type python3 and press enter to open interactive python shell. 
  • Type quit() to exit from python shell.
